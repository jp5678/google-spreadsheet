🖥️ 스프레드시트 시작하기

화면 구성을 이해하고, 시트 관리·공유·틀 고정까지 기초 환경을 완벽하게 설정합니다.

📌 구글 스프레드시트란?

행(가로줄)과 열(세로줄)로 이루어진 표 형태의 온라인 계산·기록 프로그램입니다. 인터넷만 있으면 무료로 사용할 수 있고, 변경 사항이 자동 저장되며, 여러 사람이 동시에 편집할 수 있습니다.
비교 항목종이 기록구글 스프레드시트
저장수동, 분실 위험자동 저장, 클라우드
계산수동 계산 오류 가능수식으로 자동 계산
공유복사본 필요링크 하나로 즉시 공유
수정 추적불가버전 기록으로 언제든 복원
분석시간 소요차트·피벗 테이블 즉시 생성
🏥 간호 실무 활용
활력징후 기록, 투약 관리, 낙상 위험도 평가, 실습 일정표, 근무표 등 다양한 업무에 활용됩니다.

  1. 1
    구글 계정으로 로그인 후, 브라우저 주소창에 sheets.new 입력 → Enter
    가장 빠른 방법. 즉시 새 시트가 열립니다.
  2. 2
    왼쪽 상단 "제목 없는 스프레드시트" 클릭 → 파일 이름 입력
  3. 3
    기존 파일 열기: Google Drive(drive.google.com) → 파일 더블클릭
  4. 4
    엑셀 파일 가져오기: 드라이브에 .xlsx 파일을 드래그 앤 드롭 → 마우스 우클릭 → Google 스프레드시트로 열기
💡 자동 저장: 모든 변경이 즉시 저장됩니다. 저장 버튼이 없어도 걱정 없습니다!

🗺️ 화면 구성 완전 해부

영역위치기능
메뉴바최상단파일·편집·보기·삽입·서식·데이터·도구·확장 프로그램·도움말
도구 모음메뉴바 아래자주 쓰는 기능 아이콘 (굵게, 정렬, 색상 등)
이름 상자왼쪽 상단 칸현재 선택된 셀 주소 표시 (예: B3). 직접 입력해 셀 이동 가능
수식 입력줄이름 상자 오른쪽셀에 입력된 내용·수식 표시 및 편집
열 머리글A, B, C...열 전체 선택, 너비 조정

📌 행·열 고정 (틀 고정)

스크롤해도 제목 행·열이 화면에 고정되어 항상 보이게 하는 기능입니다. 환자 데이터가 많을 때 매우 유용합니다.
  1. 1
    고정할 행 아래 셀을 선택합니다 (예: 2행 고정 → A2 선택)
  2. 2
    메뉴 보기 → 고정 → 현재 행까지(1행) 클릭
  3. 3
    굵은 회색 선이 나타나면 고정 완료! 스크롤해도 1행은 항상 보입니다.
  4. 4
    해제: 보기 → 고정 → 행 없음

✅ 데이터 유효성 검사 (입력 오류 방지)

📐 수식 기초 & 셀 참조

수식은 반드시 = (등호)로 시작합니다. 셀 주소를 참조하면 값이 바뀌어도 자동으로 재계산됩니다.
참조 유형예시특징
상대 참조A1, B3수식 복사 시 자동으로 주소 변경. 기본값.
절대 참조$A$1, $B$3복사해도 항상 같은 셀 참조. F4로 변환.
혼합 참조$A1, A$1열 또는 행 중 하나만 고정.
🏥 절대 참조 활용 예시
모든 투약량에 동일 체중(B1 셀)을 나눌 때 =$B$1을 사용하면 수식 복사 시에도 B1 셀만 참조합니다.

🔢 핵심 함수 완전 정복

① SUM — 합계

=SUM(D2:D7)    → D2~D7 합계
=SUM(D2,D5,D7)  → 지정 셀만 합계

② AVERAGE — 평균

=AVERAGE(D2:D7)  → 체온 평균

③ MAX / MIN — 최대·최소

=MAX(D2:D7)  → 최고 체온
=MIN(E2:E7)  → 최저 수축기 혈압

④ COUNT / COUNTA — 개수

=COUNT(D2:D7)   → 숫자 셀 개수
=COUNTA(C2:C7)  → 비어있지 않은 셀 개수 (텍스트 포함)

⑤ ROUND — 반올림

=ROUND(38.567, 1)  → 38.6 (소수점 1자리)
=ROUNDUP(38.1, 0)  → 39 (무조건 올림)
=ROUNDDOWN(38.9, 0) → 38 (무조건 내림)

⑥ TODAY / NOW — 오늘 날짜·현재 시간

=TODAY()  → 오늘 날짜 (자동 갱신)
=NOW()   → 현재 날짜+시간 (자동 갱신)

⑦ DATEDIF — 날짜 차이

=DATEDIF(B2, TODAY(), "Y")  → 나이 계산 (생년월일 기준)
=TODAY()-C2  → 입원 후 경과 일수
🏥 간호 실무 함수 활용
• =AVERAGE(D2:D7) → 오늘 병동 평균 체온
• =MAX(E2:E7) → 가장 높은 수축기 혈압
• =DATEDIF(B2,TODAY(),"Y") → 환자 나이 자동 계산
• =TODAY()-입원일 → 입원 일수 자동 계산

🔧 SORT · FILTER 함수 (원본 유지)

함수 방식은 원본 데이터를 건드리지 않고 다른 위치에 정렬/필터된 결과를 표시합니다.

SORT 함수

=SORT(A2:I7, 4, FALSE)

→ A2:I7 범위를 4번째 열(체온) 기준으로 내림차순 정렬한 결과를 출력

FILTER 함수

=FILTER(A2:I7, D2:D7>=38)

→ 체온이 38 이상인 행만 추출

=FILTER(A2:I7, (D2:D7>=38)*(E2:E7>=140))

→ 체온 ≥ 38 AND 수축기혈압 ≥ 140 (두 조건 동시 만족)

💡 FILTER 함수는 결과가 여러 행으로 자동 확장되는 동적 배열 함수입니다. 결과가 들어갈 공간이 비어있어야 합니다.

⚙️ 중급 함수

IF·IFS·COUNTIF/S·SUMIF/S·VLOOKUP·INDEX/MATCH·IFERROR로 데이터 분석을 자동화합니다.

🔀 IF 함수 — 조건 판단

=IF(조건, 참일때값, 거짓일때값)
수식결과
=IF(D2>=38, "발열", "정상")체온 판정
=IF(E2>=140, "고혈압", IF(E2>=120, "주의", "정상"))혈압 3단계 판정
=IF(I2<=94, "⚠️ SpO₂ 저하", "정상")산소포화도 경보

IFS 함수 — 다중 조건 (중첩 IF 대체)

=IFS(D2>=39,"고열", D2>=38,"발열", D2>=36,"정상", TRUE,"저체온")

조건이 3개 이상일 때 중첩 IF보다 훨씬 읽기 쉽습니다.

낙상 위험도 자동 평가

=IFS(J2>=13,"고위험", J2>=7,"중위험", J2>=0,"저위험")

J열 = Morse 낙상 위험도 점수

🔢 COUNTIF / COUNTIFS — 조건부 개수

=COUNTIF(D2:D20, ">=38")

→ 체온 38 이상인 환자 수

=COUNTIF(C2:C20, "김*")

→ 성이 "김"인 환자 수 (* = 와일드카드)

=COUNTIFS(D2:D20,">=38", E2:E20,">=140")

→ 발열 AND 고혈압 동시 해당 환자 수

🏥 일일 요약 보고서 자동화
발열 환자: =COUNTIF(D2:D30,">=38")  |  고혈압: =COUNTIF(E2:E30,">=140")  |  SpO₂저하: =COUNTIF(I2:I30,"<=94")

➕ SUMIF / SUMIFS — 조건부 합계

=SUMIF(C2:C20, "내과", H2:H20)

→ 내과 환자들의 호흡수(H열) 합계

=AVERAGEIF(C2:C20, "외과", D2:D20)

→ 외과 환자들의 평균 체온

🔎 VLOOKUP — 세로 방향 데이터 검색

=VLOOKUP(찾을값, 데이터범위, 열번호, 0)
첫 번째 열에서 값을 찾아 같은 행의 지정 열 값을 가져옵니다. 마지막 인수 0(FALSE) = 정확히 일치.
수식의미
=VLOOKUP("P003", A2:I7, 4, 0)P003 환자의 체온(4번째 열)
=VLOOKUP(L2, A2:I7, 3, 0)L2 셀의 환자번호로 이름 검색
=VLOOKUP(L2, 약물목록!A:D, 2, 0)다른 시트의 약물 데이터 검색
💡 VLOOKUP 한계: 첫 번째 열 기준으로만 검색 가능. 왼쪽으로는 검색 불가. 이럴 때는 INDEX+MATCH를 사용하세요.

INDEX + MATCH — VLOOKUP보다 강력한 검색

=INDEX(반환범위, MATCH(찾을값, 검색범위, 0))
=INDEX(C2:C7, MATCH("P003", B2:B7, 0))

→ B열에서 P003을 찾아 C열(이름)을 반환. 어느 방향이든 검색 가능.

🛡️ IFERROR — 오류 숨기기

=IFERROR(VLOOKUP(L2,A2:I7,4,0), "환자 없음")

→ VLOOKUP에서 오류 발생 시 "환자 없음" 표시. #N/A 오류를 깔끔하게 처리.

=IFERROR(B2/C2, 0)

→ 0으로 나누기 오류 시 0으로 표시

📝 TEXT 함수 — 값을 텍스트로 변환

=TEXT(TODAY(), "yyyy년 mm월 dd일")
=TEXT(D2, "0.0")&"℃"

→ 숫자와 단위를 합쳐 "38.5℃" 형태의 텍스트 생성

→ & 연산자로 텍스트 이어붙이기

💬 AI에게 수식 만들어달라고 하기

Gemini 패널에 수식이 필요한 상황을 설명하면 적합한 수식을 자동으로 생성합니다.
이렇게 질문하세요Gemini가 생성하는 수식
"체온이 38도 이상이면 발열, 아니면 정상으로 표시해줘"=IF(D2>=38,"발열","정상")
